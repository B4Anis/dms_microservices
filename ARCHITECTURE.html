Flow Summary

FlowTypePath (abbreviated)
Login / JWT issue Sync Browser → GW → Auth → PG
Document upload Sync Browser → GW → Documents → PG + MinIO
Document read (cache hit) Sync Browser → GW → Documents → Redis
Document read (cold) Sync Browser → GW → Documents → Redis miss → PG
Comment post / fetch Sync Browser → GW → Comments → Cassandra
Translation trigger Async Documents ══► Kafka dms.documents.uploaded
Translation execution Async Kafka ══► Translator → MinIO + Gemini API
Translation persistence Async Translator ══► Kafka dms.documents.translated ══► Documents → PG
ERP document management Sync Odoo UI → Odoo 15 (ensia_dms) → PG (odoo schema)
Metrics collection Out-of-band Prometheus → postgres-exporter → PG → Grafana
